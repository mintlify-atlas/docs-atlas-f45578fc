## What is Cossistant?

Cossistant is an open-source customer support platform built by developers, for developers. It provides a fully customizable AI support infrastructure with real-time chat widgets, headless UI components, and a complete backend—all with a code-first, API-driven philosophy.

## How It Works

<Steps>
  <Step title="Install the SDK">
    Add `@cossistant/react` or `@cossistant/next` to your project with your favorite package manager.
  </Step>
  <Step title="Configure Provider">
    Wrap your app with `SupportProvider` and set your API key.
  </Step>
  <Step title="Add the Widget">
    Drop the `<Support />` component anywhere in your app—it renders as a floating chat widget.
  </Step>
  <Step title="Customize & Deploy">
    Identify visitors, customize themes, add AI agents, and go live.
  </Step>
</Steps>

## Tech Stack

Cossistant is built with modern technologies:

- **Frontend**: React, Next.js, TypeScript, TailwindCSS
- **Backend**: Hono, tRPC, Drizzle ORM, Better Auth
- **Real-Time**: WebSockets with native browser APIs
- **Infrastructure**: Docker, PostgreSQL, Redis
- **AI**: OpenAI integration with customizable prompts
